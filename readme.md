# e-Commerce Auction Web App

An eBay-like e-commerce auction site that will allow users to post auction
listings, place bids on listings, comment on those listings, and 
add listings to a “watchlist.”

### Tech Stack
  - HTML
  - CSS
  - Python (Django)
  - Sqlite3
